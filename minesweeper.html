Minesweeper Game
In C++

Game play screen of Minesweeper game

As part of the UF Programming Fundamentals 2 course, I implemented the game Minesweeper in C++. This was my first project implemented using Object Oriented Programming and Memory Management principles and external code libraries. Specifically, the Simple and Fast Multimedia Library (SFML) was used to implement the game's user interface.

In addition to basic game play functions, the game also has buttons that allow for specific arrangements of mines to be generated for testing purposes. There is also a button that can toggle if all of the tiles on the board are visible which is also used for testing.
