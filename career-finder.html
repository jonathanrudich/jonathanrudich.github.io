Career Finder
MERN Stack Web Application

Bare-bones layout of the career view, which displays career data from the database

As part of the University of Florida Intro to Software Engineering Final Project, three other students and I are collaborating remotely to create a Web Application that can provide career advice to middle school students. We are employing Agile Scrum as our software development methodology.

When finished the app will have career search, chatbot, authentication, and admin control features among others.

We are implementing the App using the JavaScript MERN Stack (MongoDB, Express.js, React.js, Node.js) and deploying with Heroku.

I have been doing full-stack development for the project and have to date implemented RESTful CRUD APIs that interact with the MongoDB database using Express as well as have written React components on the front-end. I am now working on implementing the authentication and chatbot features with my teammates.

We are conducting remote project planning using Git/GitHub for version control, Trello for project organization and Slack and Zoom for communication.
