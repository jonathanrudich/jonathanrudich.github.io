Portfolio Optimization
Using Data Science in Python

Results of optimization analysis on portfolio of random S&P500 stocks

In finance, portfolio optimization is the process of selecting stocks so that the portfolio obtains the highest returns for the least amount of risk. This traditionally was done using Modern Portfolio Theory, a mathematical framework developed by Harry Markowitz in the 1950's. However with the advent of Big Data and Data Science, AI and Machine Learning can now be used to perform this analysis and potentially obtain better results than previously possible.

Therefore, as part of the University of Florida Intro to Data Science Final Project, I along with a teammate am performing a portfolio optimization analysis based on the historical daily time-series data from the stocks in the S&P 500 index obtained via the AlphaVantage API using both supervised and unsupervised machine learning methods.

I'm also using methods from Modern Portfolio Theory and comparing the results to see if AI really can be helpful in maximizing returns and minimizing risk when choosing stocks.

I am implementing the code in Python and developing and running the code in Google Colab, a free tool provided by Google that is useful for machine learning and data analysis.
